# Zui

**Category:** Network

**File Extensions:** `.pcap`, `.pcapng`, `.zng`

**Tags:** network-analysis, pcap, zeek

Zui is a tool for analyzing network traffic. It can read pcap and zng files and provides a powerful query language for analyzing the data.
