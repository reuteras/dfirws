# NetworkMiner

**Category:** Network

**Homepage:** <https://www.netresec.com/?page=NetworkMiner>

**Vendor:** Netresec

**License:** free edition

**Source:** HTTP

**Profiles:** Full (not included in Basic profile)

**File Extensions:** `.pcap`, `.pcapng`, `.etl`

**Tags:** network-analysis

NetworkMiner is an open source network forensics tool that extracts artifacts, such as files, images, emails and passwords, from captured network traffic in PCAP files.
