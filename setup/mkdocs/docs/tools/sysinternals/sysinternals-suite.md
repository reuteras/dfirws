# Sysinternals Suite

**Category:** Sysinternals

**Homepage:** <https://docs.microsoft.com/en-us/sysinternals/>

**Vendor:** Microsoft

**License:** [Sysinternals Software License Terms](https://docs.microsoft.com/en-us/sysinternals/license-terms)

**Source:** HTTP

**Profiles:** Full, Basic

**File Extensions:** `.exe`, `.dll`, `.sys`

**Tags:** windows, debugging, monitoring, system-administration

Sysinternals Suite is a collection of utilities for Windows.

## Tips
Sysinternals Suite is installed in .\mount\Tools\sysinternals.

## Usage
Sysinternals Suite is a collection of utilities for Windows. It includes tools for process management, file system analysis, network monitoring, and more.
