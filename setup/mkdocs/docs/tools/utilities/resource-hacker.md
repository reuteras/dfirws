# Resource Hacker

**Category:** Utilities

**Homepage:** <https://angusj.com/resourcehacker/>

**Vendor:** Angus Johnson

**License:** Freeware

**Source:** HTTP

**Profiles:** Full, Basic

**File Extensions:** `.exe`, `.dll`, `.res`

**Tags:** pe-analysis, reverse-engineering

Resource Hacker is a tool for viewing and editing resources in Windows executables.

## Tips
Resource Hacker is installed in .\mount\Tools\ResourceHacker.

## Usage
Use Resource Hacker to modify resources in PE files.
