## Tips
7-Zip is a free and open-source file archive tool.

## Usage
7-Zip is a file archive tool with both a graphical and command-line interface.

## Sample Commands
- `7z x archive.zip -oC:\ExtractedFiles`
- `7z a archive.7z C:\FilesToCompress`
- `7z x -pinfected password_protected.7z -oC:\ExtractedFiles`
