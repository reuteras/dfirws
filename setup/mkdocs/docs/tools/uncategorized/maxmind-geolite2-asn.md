## Notes
MaxMind GeoLite2 ASN database for mapping IP addresses to Autonomous System Numbers. Requires MAXMIND_LICENSE_KEY.

## Tips
Use this database with tools that support MaxMind DB format to resolve IP addresses to ASN information. Set MAXMIND_LICENSE_KEY in config.ps1 to download.

## Usage
The GeoLite2-ASN.mmdb file is downloaded from MaxMind and stored in enrichment\maxmind_current. It provides IP to ASN lookups.
