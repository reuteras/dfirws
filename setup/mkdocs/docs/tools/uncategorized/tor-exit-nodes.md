## Tips
Use these lists to identify if an IP address is a known TOR exit node. Useful for network forensics and threat intelligence.

## Usage
The exit node lists are downloaded from the Tor Project collector archive and stored in the enrichment\tor directory. Each file contains exit relay information for a specific time period.
