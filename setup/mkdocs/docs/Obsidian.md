# Obsidian

To install plugins first create a vault, for example create _Test_ in the _readwrite_ folder on the _Desktop_ and open a **Terminal**.

```shell
cd .\Desktop\readwrite\Test\.obsidian\
cp -Recurse C:\downloads\obsidian-plugins\ .
mv .\obsidian-plugins\ plugins
```

Restart Obsidian and the plugins can be enabled.
