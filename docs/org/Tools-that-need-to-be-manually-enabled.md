The following tools have to be manually enabled in *setup\config.txt* or installed with **dfirws-install.ps1**:

- [apimonitor](http://www.rohitab.com/apimonitor) - set *$WSDFIR_APIMONITOR="Yes"* to install it every time or run **dfirws-install.ps1 --apimonitor**
- [cmder](https://cmder.app/) - set *$WSDFIR_CMDER="Yes"* to install it every time or run **dfirws-install.ps1 --cmder**
- [KAPE](https://ericzimmerman.github.io/KapeDocs/#!index.md) - set *$WSDFIR_KAPE="Yes"* to install it every time or run **dfirws-install.ps1 --kape**
- [LibreOffice](https://libreoffice.org/) - set *$WSDFIR_LIBREOFFICE="Yes"* to install it every time or run **dfirws-install.ps1 --libreoffice**
- [Loki](https://github.com/Neo23x0/loki) - set *$WSDFIR_LOKI="Yes"* to install it every time or run **dfirws-install.ps1 --loki**
- [neo4j](https://neo4j.com/) - set *$WSDFIR_NEO4J="Yes"* to install it every time or run **dfirws-install.ps1 --neo4j**
- [Obisidian](https://obsidian.md/) - set *$WSDFIR_OBSIDIAN="Yes"* to install it every time or run **dfirws-install.ps1 --obsidian**
- [QEMU](https://www.qemu.org/) - set *$WSDFIR_QEMU="Yes"* to install it every time or run **dfirws-install.ps1 --qemu**
- [Visual studio code](https://code.visualstudio.com/) - set *$WSDFIR_VSCODE="Yes"* to install it every time or run **dfirws-install.ps1 --vscode**
- [Zui](https://github.com/brimdata/zui) - set *$WSDFIR_ZUI="Yes"* to install it every time or run **dfirws-install.ps1 --zui**

Programming languages:

- [GoLang](https://go.dev/) - run **dfirws-install.ps1 --golang**
- [Node](https://nodejs.org/en) - run **dfirws-install.ps1 --node**
- [Ruby](https://www.ruby-lang.org/en/) - run **dfirws-install.ps1 --ruby**
- [Rust](https://www.rust-lang.org/) - run **dfirws-install.ps1 --rust**

[Python](https://python.org/), [PHP](https://php.net/), [PowerShell](https://github.com/PowerShell/PowerShell) and [Perl](https://perl.org/) are always available in the Sandbox

Other programs:

- [Autopsy](https://www.sleuthkit.org/autopsy/) - run **dfirws-install.ps1 --autopsy**
- Bash extra tools - run **dfirws-install.ps1 --bashextra**
