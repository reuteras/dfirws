[Ghidra](https://github.com/NationalSecurityAgency/ghidra) is available. Extra extensions can be found under *C:\Tools\ghidra_extensions*. Currently the following extensions can be added:

- [Ghidra_GolangAnalyzerExtension](https://github.com/mooncat-greenpy/Ghidra_GolangAnalyzerExtension)
- [Ghidrathon](https://github.com/mandiant/Ghidrathon)

Some preferences for Ghidra is done via the default [example-customize.ps1](https://github.com/reuteras/dfirws/blob/main/local/example-customize.ps1) that is copied to *local/customize.ps1*.

The diffing tool [Ghidriff](https://github.com/clearbluejar/ghidriff) uses Ghidra to diff binaries. For more information read this blog post, [Ghidriff: Ghidra Binary Diffing Engine](https://clearbluejar.github.io/posts/ghidriff-ghidra-binary-diffing-engine/)

## Versions

Currently the versions installed are 10.4 with the extensions mentioned above. The latest version is also installed, currently 11.0.

## Ghidrathon and capa

Ghidrathon is available to install and use in the sandbox. An introduction can be found in this Mandiant blog post, [Riding Dragons: capa Harnesses Ghidra](https://www.mandiant.com/resources/blog/capa-harnesses-ghidra) ([capa-harnessess-ghidra.pdf](./resources/14331165/capa-harnessess-ghidra.pdf)).
